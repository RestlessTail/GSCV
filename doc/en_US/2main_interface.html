Main interface

Main interface contains of three parts: tool bar (left), viewport (middle) and tool panel (right).

Tool bar

The function of buttons in the tool bar are demonstrated in the following table.

Icon Function
Zoom in.
Zoom out.
Save current viewport as image.
Copy current viewport to clipboard.

Viewport

The viewport displays the result of painting. You can drag the viewport with the mouse.

Note: The behavior of viewport differs according to performance settings. In consideration of devices with weaker hardware, a strict resource control is applied by default. When dragging the viewport, the viewport do not update until the mouse released. If you want the viewport to update when dragging the viewport, you can change its behavior in settings.
