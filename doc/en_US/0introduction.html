Introduction

GSCV (Graphical Single Cell Viewer) is a tool for visualizing single cell data. The program is designed for PC and laptop, which highlights low memory usage and high speed of painting. The program is able to paint feature plot without any script, which is suitable for researchers who are not familar with R script or linux server.
