Custom color scheme

Open custom panel

Click Visualize-Color scheme, and the custom panel is activated.

Custom gradient color

Gradient mode is used when visualizing expression data or continuous meta data. You can choose two or three colors (by selecting the "Bicolored" radio button or "Tricolored" radio button), and the program will calculate the color of each cell automatically.

To change color, click the color piece and select a new color in the dialog. The line edit is read-only, and you can copy the hex color code, but you can't specify the color code directly in the line edit.

Custom color set for categorical data

Categorical mode is used only when visualizing categorical meta data. It is required to determine a color set before drawing the plot. The default color set contains of six colors. To change color, click the color piece and select a new color in the dialog. You can also adjust the order of colors, add new color, and delete existing color by using the buttons at the right of the list box.

The function of buttons on the right are demonstrated in the following table.

Button Function
Add Add a new color (black by default) at the end of the color set.
Remove Remove selected color.
Up Swap selected color and the color ahead of it.
Down Swap selected color and the color behind of it.

Custom color of inactive cells

Inactive cells are cells that are not colored according to expression data or meta data. For example, when opening a new archive or clicking "Clear" in cell color panel, the cells are colored as inactive cell.

To change color, click the color piece and select a new color in the dialog. The line edit is read-only, and you can copy the hex color code, but you can't specify the color code directly in the line edit.

Custom background

To change background color, click the color piece and select a new color in the dialog. The line edit is read-only, and you can copy the hex color code, but you can't specify the color code directly in the line edit.

Close this panel

Click the "Close" button, and the panel is closed.
