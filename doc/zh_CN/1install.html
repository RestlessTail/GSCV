安装

需求

  • Windows 10 或 X11 （暂不支持Mac OS）
  • OpenGL 3.3或更高

从源代码编译

要从源代码编译GSCV，您需要以下依赖项：

  • Qt 6
  • zlib

Qt Creator

Linux用户可以用Qt Creator打开GSCV.pro并直接编译。Windows用户需要自定义GSCV.pro中的zlib路径。

运行 “lrelease GSCV.pro” 命令，并将“icon/”, “shaders/”, “doc/”, “language/”拷贝到程序目录下。对于Windows用户，还需要运行 “windeployqt build” 命令。

Cmake

首先，下载代码仓库并解压（如果有需要的话）。

  1. wget https://
  2. tar xzvf ... （如果有需要的话）
  3. cd GSCV/

使用Cmake构建工程，并编译（Windows用户需要手动准备zlib并修改CMakeLists.txt）。

  1. mkdir build
  2. cd build
  3. cmake ..
  4. make
  5. cd ..

运行 “lrelease GSCV.pro” 命令，并将“icon/”, “shaders/”, “doc/”, “language/”拷贝到程序目录下。对于Windows用户，还需要运行 “windeployqt build” 命令。
