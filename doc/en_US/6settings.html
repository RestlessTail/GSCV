Performance

Repaint instantly

When the check box selected, the viewport updates regularly while dragging viewport or sliders. Otherwhise the viewport updates only when mouse is released.

The affected action are as follows:

  • Drag viewport
  • Adjust point size
  • Zoom in & zoom out (by using a slider)

Note: Selecting "Repaint instantly" could increase GPU usage significantly. In consideration of devices with weaker hardware, it is disabled by default.

Viewport maximum frame rate

Set the maximum frame rate when dragging viewport or slider. If "Repaint instantly" not selected, this option won't change the program's behavior.
